Read Me file for Random Fantasy Character Generator
**************************************************
This is a simple, terminal based program that generates
stats for two fantasy characters.

Each character's stats are randomized.

Vit - Vitality
The total health points.

Pow - Physical Power
The damage the character deals with physical attacks.

Int - Magical Power
The damage the character deals with magical attacks.

Dex - Dexterity
Affects dodging ability.

Will - Magical Defense 
How resistant the character is to magical attacks

Con - Constitution
How resistant the character is to physical attacks
*************************************************
*The name of the character affects the stats.

*The character with the lower vitality strikes first.
